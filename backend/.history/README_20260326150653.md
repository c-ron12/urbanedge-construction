# 🛠️ Urbanedge Construction - Backend API

## 📌 About the Project

This is the backend API for the Urbanedge Construction project, built using Laravel. It provides RESTful APIs for managing services, projects, articles, members, and testimonials.

---

## ⚙️ Tech Stack

* Laravel
* MySQL
* REST API

---

## ✨ Features

* CRUD operations for:

  * Services
  * Projects
  * Articles
  * Members
  * Testimonials
* Image upload handling
* API authentication (Bearer Token)
* Structured JSON responses

---

### 🔹 Planned Features

*** Move to Trash & Restore** for soft deletion of records:
  * Services, Projects, Articles, Team Members, and Testimonials
  * Instead of hard deletion, records will be marked as "deleted" (`deleted_at`) and can be restored later.
  * This feature will use Laravel's `SoftDeletes` trait and corresponding API endpoints for restore.

## 🔌 API Usage

The API is consumed by a React frontend application.

---

## 🛠️ Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
