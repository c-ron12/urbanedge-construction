# 🛠️ Urbanedge Construction - Backend API

## 📌 About the Project

This is the backend API for the Urbanedge Construction project, built using Laravel. It provides RESTful APIs for managing services, projects, articles, members, and testimonials.

---

## ⚙️ Tech Stack

* Laravel
* MySQL
* REST API

---

## ✨ Features

* CRUD operations for:

  * Services
  * Projects
  * Articles
  * Members
  * Testimonials
* Image upload handling
* API authentication (Bearer Token)
* Structured JSON responses

---

## 🔌 API Usage

The API is consumed by a React frontend application.

---

## 🛠️ Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
